Запускать 1.pl следует подавая ему в качестве аргументов список файлов через пробел (в файлах должна содержаться целые неотрицательные числа). В случае если суммарный размер файлов чересчур велик, программа прекратится когда обнаружит это.При наличии в файлах нечисловой информации будет выдано предупреждение. При большом размере файла будет выдано предупреждение о медленной работе программы.
